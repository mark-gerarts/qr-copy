FAQ

I can't make a connection, what is going wrong?

There are multiple things that could be going wrong:

  • If you are connecting manually, you might have entered the wrong ID. Make sure the ID matches that of your other device.
  • P2P communication is set up using 443. If you have a firewall enabled, make sure this port is not blocked.
  • Some browser addons might block P2P connection, primarily adblockers. Try disabling all of your addons.
  • You might be using an unsupported browser. For a list of supported browsers, see this page.
  • You could be behind a symmetric NAT. In this case there is not much you can do.

If you think none of the above apply, consider reporting an issue.

How does this work?

Ultimately this application is based on WebRTC, a web standard for real-time communication between devices. We use PeerJS, which provides a library to easily implement P2P communication and provides a server to initialize the connection. If you enjoy this application, consider donating to PeerJS!

What are some example use cases?

I primarily made this application to quickly transfer files and text to and from my iPhone without installing any applications like iTunes, but it's useful for sending files between two phones as well. If you have any other specific use cases, I'd love to learn about them!

How is this built?

The application is made using the following non-inclusive list of technologies:

  • WebRTC
  • PeerJS
  • Svelte
  • Spectre.CSS
